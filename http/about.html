資訊之芽是什麼東西？

資訊之芽是一個資訊領域的推廣計畫，透過提供正規教育與補習班以外的課程，讓更多對資訊領域有潛在 / 明顯興趣的同學能夠有機會和資訊來場相親！

資訊之芽提供什麼課程？

目前我們提供的有教授...

  1. 基礎 C 語言的「C 語法班」
  2. 基礎 Python 語言的「Python 語法班」
  3. 解題方法的「算法班」(須具備 C/C++ 語法基礎才能報名)

哪裡上課何時上課？

我們預計下學期每週末會在臺灣大學/清華大學校內上課，時間與詳細地點請見這裡。

誰能參加資訊之芽？

目前資訊之芽的主要招生對象為高中一到三年級與同等學力的在學學生，非此類對象者仍可以嘗試報名，但會有較低的優先序喔！
